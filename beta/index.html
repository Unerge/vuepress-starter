欧拉公式（Euler's formula）是复分析中的一个重要公式，由瑞士数学家莱昂哈德·欧拉（Leonhard Euler）在18世纪提出。这个公式揭示了复数指数函数与三角函数之间的深刻联系，其形式为：

eix=cos⁡(x)+isin⁡(x) e^{ix} = \cos(x) + i\sin(x) eix=cos(x)+isin(x)

其中：

  • eee 是自然对数的底（约为2.71828）。
  • iii 是虚数单位，满足 i2=−1i^2 = -1i2=−1。
  • xxx 是实数。

这个公式可以通过多种方法推导出来，包括泰勒级数展开法和微分方程法。下面我会详细解释这两种推导方法。

1. 泰勒级数展开法

泰勒级数是一种将函数表示为无穷级数的方法。对于指数函数 eixe^{ix}eix、余弦函数 cos⁡(x)\cos(x)cos(x) 和正弦函数 sin⁡(x)\sin(x)sin(x)，它们的泰勒级数分别为：

eix=∑n=0∞(ix)nn! e^{ix} = \sum_{n=0}^{\infty} \frac{(ix)^n}{n!} eix=n=0∑∞​n!(ix)n​

cos⁡(x)=∑n=0∞(−1)nx2n(2n)! \cos(x) = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!} cos(x)=n=0∑∞​(2n)!(−1)nx2n​

sin⁡(x)=∑n=0∞(−1)nx2n+1(2n+1)! \sin(x) = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} sin(x)=n=0∑∞​(2n+1)!(−1)nx2n+1​

将 eixe^{ix}eix 的泰勒级数展开：

eix=1+ix+(ix)22!+(ix)33!+(ix)44!+⋯ e^{ix} = 1 + ix + \frac{(ix)^2}{2!} + \frac{(ix)^3}{3!} + \frac{(ix)^4}{4!} + \cdots eix=1+ix+2!(ix)2​+3!(ix)3​+4!(ix)4​+⋯

=1+ix+i2x22!+i3x33!+i4x44!+⋯ = 1 + ix + \frac{i^2 x^2}{2!} + \frac{i^3 x^3}{3!} + \frac{i^4 x^4}{4!} + \cdots =1+ix+2!i2x2​+3!i3x3​+4!i4x4​+⋯

由于 i2=−1i^2 = -1i2=−1， i3=−ii^3 = -ii3=−i， i4=1i^4 = 1i4=1 等，可以进一步简化：

=1+ix−x22!−ix33!+x44!+⋯ = 1 + ix - \frac{x^2}{2!} - \frac{ix^3}{3!} + \frac{x^4}{4!} + \cdots =1+ix−2!x2​−3!ix3​+4!x4​+⋯

将实部和虚部分开：

=(1−x22!+x44!−⋯ )+i(x−x33!+x55!−⋯ ) = \left(1 - \frac{x^2}{2!} + \frac{x^4}{4!} - \cdots \right) + i\left(x - \frac{x^3}{3!} + \frac{x^5}{5!} - \cdots \right) =(1−2!x2​+4!x4​−⋯)+i(x−3!x3​+5!x5​−⋯)

观察到这些分别是余弦函数和正弦函数的泰勒级数：

eix=cos⁡(x)+isin⁡(x) e^{ix} = \cos(x) + i\sin(x) eix=cos(x)+isin(x)

2. 微分方程法

考虑复指数函数 f(x)=eixf(x) = e^{ix}f(x)=eix，我们知道它满足以下微分方程：

f′(x)=ieix=if(x) f'(x) = i e^{ix} = if(x) f′(x)=ieix=if(x)

假设 f(x)=cos⁡(x)+isin⁡(x)f(x) = \cos(x) + i\sin(x)f(x)=cos(x)+isin(x)，则：

f′(x)=−sin⁡(x)+icos⁡(x) f'(x) = -\sin(x) + i\cos(x) f′(x)=−sin(x)+icos(x)

比较两边，我们看到：

i(cos⁡(x)+isin⁡(x))=−sin⁡(x)+icos⁡(x) i(\cos(x) + i\sin(x)) = -\sin(x) + i\cos(x) i(cos(x)+isin(x))=−sin(x)+icos(x)

将右边展开：

icos⁡(x)−sin⁡(x)+i2sin⁡(x)=−sin⁡(x)+icos⁡(x) i\cos(x) - \sin(x) + i^2 \sin(x) = -\sin(x) + i\cos(x) icos(x)−sin(x)+i2sin(x)=−sin(x)+icos(x)

因为 i2=−1i^2 = -1i2=−1，所以 i2sin⁡(x)=−sin⁡(x)i^2 \sin(x) = -\sin(x)i2sin(x)=−sin(x)，这两个表达式相等。因此，

eix=cos⁡(x)+isin⁡(x) e^{ix} = \cos(x) + i\sin(x) eix=cos(x)+isin(x)

欧拉公式的几何解释

欧拉公式在复平面上有一个重要的几何解释。考虑复平面上的一个点，其极坐标表示为 reiθre^{i\theta}reiθ，其中 rrr 是模长，θ\thetaθ 是辐角。根据欧拉公式，这个点的笛卡尔坐标表示为：

(rcos⁡(θ),rsin⁡(θ)) (r\cos(\theta), r\sin(\theta)) (rcos(θ),rsin(θ))

因此，复数 eixe^{ix}eix 可以看作是一个单位圆上的点，其角度为 xxx 弧度。

欧拉恒等式

欧拉公式在 x=πx = \pix=π 时，得到著名的欧拉恒等式：

eiπ+1=0 e^{i\pi} + 1 = 0 eiπ+1=0

这被认为是数学中最美丽的公式之一，因为它将五个最重要的数学常数 e,i,π,1,0e, i, \pi, 1, 0e,i,π,1,0 通过简单的关系连接在一起。

欧拉公式不仅在纯数学中有重要应用，也在物理学、电工程学和其他工程领域中具有广泛的应用。它为许多复杂问题提供了简洁而强大的工具。
